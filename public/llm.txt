# Kenly - AI-Powered Client Onboarding Platform

## Overview
Kenly is a professional client portal and onboarding automation platform designed for agencies and service-based businesses. It streamlines the client intake process by providing a centralized workspace where clients can complete forms, upload files, sign contracts, and track project progress.

## Purpose
Kenly solves the problem of chaotic client onboarding by replacing scattered tools (email, Google Forms, Dropbox, Slack) with a single branded platform. It reduces the time agencies spend on manual follow-ups and administrative work, allowing them to focus on billable projects.

## Core Features

### Client Onboarding
- **Custom Intake Forms**: Build branded, customizable forms tailored to specific workflows
- **Smart File Uploads**: Automated file collection and organization with secure storage
- **Automated Follow-ups**: Intelligent reminder system that sends gentle notifications to clients
- **Progress Tracking**: Real-time dashboards showing onboarding completion status

### Collaboration & Communication
- **Client Portal**: Branded workspace where clients access all project information
- **Task Management**: Assign and track tasks for both agency team and clients
- **Meeting Scheduling**: Integrated calendar for booking client meetings
- **File Sharing**: Secure document storage and sharing with version control

### Automation & AI
- **AI Client Summaries**: Automatically generates insights from client form responses
- **Workflow Automation**: Trigger actions based on form submissions and status changes
- **Email Notifications**: Automated confirmation emails and status update notifications
- **Reminder System**: Scheduled follow-ups for incomplete submissions

### Business Tools
- **Contract Management**: Upload, share, and track contract signing status
- **Invoice Tracking**: Monitor payment status and due dates
- **Team Collaboration**: Multi-user access with role-based permissions (Owner, Admin, Member)
- **Branding Customization**: Add logo, custom colors, and fonts to match agency identity

## Target Audience

### Primary Users
- **Creative Agencies**: Design studios, branding agencies, marketing firms
- **Digital Agencies**: Web development, SEO, and digital marketing companies
- **Consulting Firms**: Business consultants requiring structured client intake

### User Roles
- **Agency Owners/Admins**: Manage organization settings, team members, and billing

## Technical Information

### Technology Stack
- Frontend: React, TypeScript, Vite
- UI Framework: Tailwind CSS, shadcn-ui components
- Backend: Supabase (PostgreSQL database, Authentication, Storage, Edge Functions)
- Email Service: Resend for transactional emails
- Payment Processing: LemonSqueezy for subscription management

### Key Capabilities
- Real-time data synchronization
- Secure file storage with access control
- Row-level security (RLS) for data protection
- Responsive design (mobile, tablet, desktop)
- Dark mode support
- Usage-based limits tied to subscription plans
